About the project

This project is about creating a mobile app and website for WorldSurf. The Web Interface and Usability course teaches students the concepts and strategies needed to create successful web interfaces. In this course, students examine the foundation of creating logical, intuitive, and clear web interfaces used by companies worldwide. This course examines design principles relating from usability, visualization, and functionality constructs.
